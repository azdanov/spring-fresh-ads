About

This app is powered by Java. The backend is made in Spring, Spring Boot, Spring Data JPA and secured by Spring Security. The frontend is created with Thymeleaf and Bootstrap 5 with minimal JavaScript, apart from Stripe payment integration.

If you have any questions, you can write to me anton.zdanov@pm.me or check out the source code at github.com/azdanov/spring-fresh-ads
